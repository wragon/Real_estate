# 일본 부동산 버블경제와 한국 부동산의 상관관계
* 한국 부동산 데이터 시각화 경진대회

## 개요
우리나라의 경제 10년 후는 지금의 일본경제라는 말이 있다.  
집값도 많이 상승했고 빚내서 주식을 사라는 말도 나왔던 상황에서  
우리나라는 일본과 비교하여 얼마나 집값이 폭등했고 경제 상황은 어떤 지 확인해보았다.  

## 사용 기술
- Pandas
- DataFrame
- Matplotlib
- LSTM
- Regression Analysis
